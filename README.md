# Flood-Safety Website

A single-page website created for the Flood-Safety hackathon, designed to teach essential flood safety tips.

## Team

- **Team Name:** Safety Specialists
- **School:** California High
- **Grade:** 11

## Sections

1. **Before a Flood** - Preparation tips and emergency kit checklist
2. **During a Flood** - Safety actions to take during flooding
3. **After a Flood** - Recovery steps and cleanup safety

## Features

- Responsive design works on mobile and desktop
- Color-coded sections for easy navigation
- Emergency contact information
- Practical safety checklists
- Accessible navigation menu

- `index.html` - Complete website with embedded CSS
